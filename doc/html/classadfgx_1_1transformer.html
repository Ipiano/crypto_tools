Detailed Description

The adfgx transformer.

Construct this object using a key for the adfgx cipher, and use it to encrypt and/or decrypt multiple texts using that key

std::string adfgx::transformer::decrypt ( const std::string &  cipher)

Decrypts a string using the ADFGX cipher and the loaded key.

Decrypts an ADFGX ciphertext using the default substitution matrix. All letters decrypted will be lower-case values in the range a-z. Any characters in the ciphertext which are not from [adfgx] will be removed.
