Core frequency counter.

Processes all data provided between the start and end input iterators. Each value is mapped to a value in the output sequence using the map function. The data is accessed by adding the index to the output iterator, and that value is incremented with the increment function.

Convenience function to count frequency in a string.

Processes all data provided between the start and end input iterators. The output iterator's sequence should have a size of at least 255. Characters are mapped to the range 0-255 by their ASCII value. If the function is not case sensitive, then upper-case values are mapped to their lower-case equivalent's ASCII value. The data is accessed by adding the index to the output iterator, and that value is incremented with the increment function.

Convenience function to count frequency in a text stream.

Processes all data provided by the input stream. Data is read using std::getline. The output iterator's sequence should have a size of at least 255. Characters are mapped to the range 0-255 by their ASCII value. If the function is not case sensitive, then upper-case values are mapped to their lower-case equivalent's ASCII value. The data is accessed by adding the index to the output iterator, and that value is incremented with the increment function.
