Typedef for a permutation used by the D\+ES. 

Permutations are used in the D\+ES to transform values bitwise. They may result in more or fewer bits than the input.

An example permutation \mbox{[}0, 2, 31, 23\mbox{]} would take a 32-\/bit number, and output a 4-\/bit number consisting of bits 0, 2, 31, and 23 of the input
