Converts a continued fraction to a real value. 

Generates a double value from a continued fraction form by using the formula $ x = i + 1/x $ for each value in the continued fraction in reverse order.

By default, all values will be used once. If repetitions is set to n (n != 0), then the first n times the loop hits the second value in the input, the loop will jump back to the end.

For example\+: \mbox{[}1, 2, 3, 2\mbox{]} with 0 repetitions (default) will process 2, 3, 2, 1. With 2 repetitions, it will process 2, 3, 2, 2, 3, 2, 2, 3, 2, 1.

Converts a continued fraction to approximate rationals. 

Generates all p/q approximations of the continued fraction using the following formulas
\begin{DoxyItemize}
\item $ p(k) = a(k) * p(k-1) + p(k-2) $
\item $ q(k) = a(k) * q(k-1) + q(k-2) $ Where
\item $ p(-2) = 0, p(-1) = 1 $
\item $ q(-2) = 1, q(-1) = 0 $
\end{DoxyItemize}

By default, all values will be used once. If repetitions is set to n (n != 0), then the first n times the loop hits the last value in the input, the loop will jump back to the second value.

For example\+: \mbox{[}1, 2, 3, 2\mbox{]} with 0 repetitions (default) will process 1, 2, 3, 2. With 2 repetitions, it will process 1, 2, 3, 2, 2, 3, 2, 2, 3, 2

Converts a real value to a continued fraction. 

Takes a real value and converts it to a continued fraction, represented as \mbox{[}i; f, f, f...\mbox{]} where i is the integer part of the fraction, and f1 -\/ fn are the integer values added on the layers of the fraction. The first value in the array returned is i, and the remaining values are f1 -\/ fn, in order.

The result may contain only an integer value

If the result has a cycle, it will not be detected. If you need cycle detection in a simple square root, use \hyperlink{namespacecryptomath_1_1continuedfractions_ae623004528a5e561255f468807116656}{sqrttocf}

Converts a simple square root to a repeating continued fraction. 

Finds the continued fraction of sqrt(n) as an integer i and repeating continued fraction coefficients. The first value in the returned continued fraction is the integer portion, and the remaning values will repeat infinitely. The end of the period is the first value in the repeating fraction a such that $ a == 2*i $
