\subsection{Detailed Description}
The affine transformer. 

Construct this object using some $ \alpha, \beta $ and alphabet for an affine cipher, and use it to encrypt and/or decrypt multiple texts using that pair and alphabet

Construts an affine transformer with a specific $ \alpha, \beta $ and alphabet. 

Initializes the transformer with a key pair and alphabet. If case\+Sensitive is false, then all upper-\/case text will be treated as lower-\/case during the transform (but it will be outputted in upper-\/case); in this case, all upper-\/case values in the alphabet will be changed to lower-\/case.

Constraints
\begin{DoxyItemize}
\item The alphabet must be non-\/empty
\item The alphabet must not contain duplicates
\item $ \alpha $ must be coprime with $ n $
\end{DoxyItemize}

Decrypts a text with the affine transform using the loaded key and alphabet. 

If the transformer is case-\/insensitive, then upper-\/case values will be transformed as if they were lower-\/case, but the resulting value will be outputted upper-\/case (if it is alphabetic). This may result in loss of some capitalization from the original message if the alphabet contains non-\/alphabetic characters.

Any characters not in the alphabet will be unchanged
